Contamination Filter

## Up and running

```sh
$ git clone https://github.com/COMBAT-TB/confil.git
...
$ cd confil
$ virtualenv envname
$ source envname/bin/activate
$ pip install -r requirements.txt
$ python setup.py install
$ confil --help
```
